What is Lua?

Lua is a scripting language that prides itself on its speed. The language is embeddable and supports procedural, object-oriented, functional, and data-driven programming as well as data description. The language also has variants that you can choose to use.

History of Lua

Lua was created in 1993 by Roberto Ierusalimschy, Luiz Henrique de Figueiredo, and Waldemar Celes, members of the Computer Graphics Technology Group (TeCGraf) at the Pontifical Catholic University of Rio de Janeiro (PUC-Rio) in Brazil. Initially designed to meet the demand for a lightweight, embeddable scripting language in industrial applications, Lua's first use case was for data entry tools at Petrobras, Brazil's state-owned oil company.

The language evolved from SOL (Simple Object Language) and DEL (Data-Entry Language), two domain-specific languages developed at TeCGraf. Lua 1.0 was released in July 1993, featuring a simple syntax inspired by Modula and Scheme. By version 3.0 in 1997, Lua introduced support for functions as first-class values and anonymous functions.

Lua 5.0, released in 2003, marked a major milestone with the introduction of a standalone interpreter and a new virtual machine. The language gained global traction in gaming and embedded systems throughout the 2000s. Version 5.3 (2015) added integers, bitwise operations, and UTF-8 support. Today, Lua remains actively maintained by its original creators, with version 5.4.6 released in 2023.

Where is Lua used?

Lua is used in several industrial applications and especially with embedded systems. It is currently the leading scripting language for games, even winning the Front Line Award in 2011 from the Game Developers Magazine. Even for games or apps primarily coded in other languages, it's common to see them use Lua for mods or extensions. Examples include:

  • Adobe's Photoshop Lightroom
  • World of Warcraft
  • Angry Birds
  • Roblox
  • Garry's Mod
  • Project Zomboid
  • Factorio
  • Friday Night Funkin'
  • MySQL Workbench
  • Scrap Mechanic
  • Warframe

Lua's Advantages

Speed

Several benchmarks show Lua as the fastest interpreted scripting language. It is also known that an aspiration of scripting languages is to be "as fast as Lua" as it has built a reputation on performance. Significant portions of larger applications have been written in Lua. For additional speed, developers recommend using LuaJIT, an independent implementation that uses a just-in-time compiler.

Portability

The language is distributed in a small package that builds out-of-the-box on any platform with a standard C compiler. This means it runs across Unix variants, Windows, Android, iOS, BREW, Symbian, and Windows Phone. It's even used on embedded microprocessors like ARM and Rabbit, including platforms like Lego Mindstorms.

Embeddable

Lua has a small footprint and well-documented API that allows seamless integration with code in other languages. It can be extended with libraries from other languages and has been used to extend programs written in other scripting languages.

Powerful

A fundamental concept behind Lua is using meta-mechanisms to implement features instead of providing features directly. While not a pure object-oriented language, it uses meta-mechanisms to implement classes and inheritance, allowing flexible extension of semantics in unconventional ways.

Small

Adding Lua to an application doesn't significantly increase its size (tarball: 365K compressed, 1.3M uncompressed). The interpreter is 279K and library 464K, contributing to both efficiency and security.

Free

Lua is free open-source software distributed under the MIT license, available for any purpose at no cost.
