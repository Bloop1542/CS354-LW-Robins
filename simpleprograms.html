Factorial Function

Calculates the factorial of a number and prints the result.

ðŸ“¥ How to Run

  1. Save the code as factorial.lua
  2. Run it in the terminal using:
lua factorial.lua

function factorial(num)
    local product = 1
    for i = 1, num do
        product = product * i
    end
    print("Factorial of " .. num .. " is " .. product)
end

factorial(6)

FizzBuzz

Prints numbers from 1 through N, replacing multiples of 3 with "fizz", 5 with "buzz", and both with "fizzbuzz".

ðŸ“¥ How to Run

  1. Save the code as fizzbuzz.lua
  2. Run it using:
lua fizzbuzz.lua

function fizzbuzz(max)
    local sequence = {}
    for i = 1, max do
        local temp = ""
        if i % 3 == 0 then 
            temp = temp .. "fizz"
        end
        if i % 5 == 0 then
            temp = temp .. "buzz"
        end
        if temp == "" then 
            temp = temp .. i
        end
        sequence[i] = temp
    end
    return sequence
end

print(table.concat(fizzbuzz(15), ", "))

Even Number Printer

This program prints all even numbers from 1 to a specified maximum.

ðŸ“¥ How to Run

  1. Save the code as even.lua
  2. Run it using:

  -- function that finds all the even numbers in a given range from 1 to max
  function even(max)
    local sequence = {}
    for i = 1, max do
      if i % 2 == 0 then
        sequence[i/2] = i
      end
    end
    return sequence
  end
    
  print(table.concat(even(16), ", "))

  function isPrime(number) 
      if number <= 1 then
          return false
      end
      for i = 2, math.floor(math.sqrt(number)) + 1 do
          if number % i == 0 then
              return false
          else 
              return true
          end
      end
  end
