Installing Lua

Windows Installation

How to Install

  1. Download the Windows binaries from lua.org/ftp
  2. Add Lua to your PATH environment variable
  3. Verify installation in Command Prompt:
lua -v

Linux Installation

How to Install

  1. For Debian/Ubuntu-based systems:
sudo apt-get install lua5.4
  1. For Fedora/RHEL-based systems:
sudo dnf install lua

macOS Installation

How to Install

  1. Using Homebrew:
brew install lua
  1. Verify installation:
lua -v

Lua Translators

LuaJIT (Just-In-Time Compiler)

How to Install

  1. Download from luajit.org
  2. Build from source:
make && sudo make install
  1. Verify installation:
luajit -v

LuaRocks (Package Manager)

How to Install

  1. Windows: Use the standalone installer
  2. Linux/macOS:
wget https://luarocks.org/releases/luarocks-3.9.2.tar.gz
tar zxpf luarocks-3.9.2.tar.gz
cd luarocks-3.9.2
./configure && make && sudo make install
  1. Verify installation:
luarocks --version

Verifying Your Setup

Create a test file hello.lua with:


print("Hello, Lua!")

Run it with:


lua hello.lua

For LuaJIT:


luajit hello.lua
